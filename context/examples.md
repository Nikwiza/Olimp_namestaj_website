## What to Extract

When screenshotting references, note:
- Hero section structure and CTA placement
- Gallery/portfolio grid patterns
- How trust signals (years, testimonials) are displayed
- Typography hierarchy and spacing
- Color palette usage
- Mobile navigation patterns
- Contact section layouts
